> Olá, este projeto é um teste de aprendizado do Git.

> git init -> cria repositório
> git add -> arquivo -> stage
> git commit
> git remote -> conexão do repositório local com repositório do Git

> README.md -> new file -> adiciona os arquivos do projeto

> git add .
> git status
> git commit -m mesage
> git push origin main para mandar para o Git 
  - evitar ficar editando lá e aqui, se der erro de entre as versões, mandar:
  - git status
  - corrigir o que precisa
  - git add .
  - git commit -m "resolve merge conflit" 
  - git push origin main

> git checkout - sai da main e entra na branch
> merge junta com a nova branch
  - em casos de erro no merge: git pull origin main --no-rebase + ESC + :q!
  - seguir com git push origin main

> Para pegar o repositório do Github e puxar arquivos do proejeto para a minha máquina, para abrir em outras, cria pasta, git bash e git clone
